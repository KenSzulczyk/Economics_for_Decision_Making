# 📘 Homework 1: Data Visualization and Economic Intuition with Python

This assignment introduces students to data visualization using Python. You will explore real-world economic data, create visualizations, and connect the results to economic theory.

The goal is simple: learn how to see patterns in data before jumping into complex models.

## 📊 What We Will Learn

By completing this homework, we will:

* Create and work with datasets in Python
* Visualize data using boxplots and scatter plots
* Interpret distributions using median and interquartile range (IQR)
* Understand the relationship between price and demand
* Analyze financial time series data using moving averages
* Identify bull and bear markets in stock market data

## 🧠 Economic Concepts

This assignment connects coding to key economic ideas:

* Law of Demand → Higher prices tend to reduce consumption
* Data dispersion → Real-world data is noisy and influenced by many factors
* Multivariate thinking → Prices alone do not explain behavior
* Financial trends → Moving averages help identify long-term vs short-term trends

## 🧪 Assignment Structure

Part 1: Coffee Market Data

We will:

* Create a dataset manually in Python
* Generate boxplots to understand distributions
* Create a scatter plot of coffee price vs consumption
* Add a regression line to visualize the relationship

👉 Focus: Understanding variability and economic relationships

Part 2: Financial Time Series (Hang Seng Index)

You will:

* Load real stock market data
* Plot the Hang Seng Index over time
Add:
* 50-day moving average (short-term trend)
* 200-day moving average (long-term trend)
Highlight:
* Bull markets (green)
* Bear markets (red)
